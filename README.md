# ООО "Облачные Технологии" - Тестовое задание

## Проект "Форма регистрации"

Данный проект является тестовым заданием для ООО "Облачные Технологии" и представляет собой форму регистрации. Цель проекта — продемонстрировать навыки разработки и стилизации React-компонентов, а также взаимодействия с пользовательским вводом и валидацией данных.

### Технологии:

- **React**: Используется для построения интерактивных UI компонентов.
- **Styled-Components**: Используется для стилизации компонентов.
- **React Hook Form**: Помогает с формами и их валидацией.
